# 更新日志

## 1.0.0 (2024-01-01)

### 新增

- ✨ 基于 Ant Design Vue 的二次封装组件库
- ✨ 提供 `AButton` 按钮组件
- ✨ 提供 `AInput` 输入框组件
- ✨ 完整的 TypeScript 类型支持
- ✨ Monorepo 架构
- ✨ VitePress 文档站点
- ✨ 工具函数库 `@my-vue/utils`
  - `debounce` 防抖函数
  - `throttle` 节流函数
  - `deepClone` 深拷贝
  - `formatDate` 格式化日期
  - `randomString` 生成随机字符串
  - `isEmpty` 判断是否为空
  - `isMobile` 判断是否为移动设备

### 特性

- 🎨 基于 Ant Design Vue 4.x
- 🚀 基于 Vite 构建
- 📦 支持按需引入
- 💪 完整的 TypeScript 支持
- 🛠️ 业务化组件封装
- 📚 完整的文档和示例
